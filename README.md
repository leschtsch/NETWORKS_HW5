# Запуск
`RUNME.sh` или вручную через CMake
Программы будут в `bin/`

Можно запустить без параметров, пути к сертификатам по умолчанию прописаны.
Файл сертификата только один, его нужно указать и серверу, и клиенту, чтобы он ему доверял.
